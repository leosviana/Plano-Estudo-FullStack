Módulos: Import/Export

Antes colocavamos todo o código em apenas um arquivo .js

Agora podemos dividir o código em vários arquivos e conectar eles com import/export.

Estrutura de projeto Exemplo:

- index.html

- js/

-- main.js

-- funcoes.js

-- usuario.js

**Podemos também exportar uma única função ou classe principal de um arquivo.
